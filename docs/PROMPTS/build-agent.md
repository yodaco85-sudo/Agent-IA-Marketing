# PROMPT — Lancer / reprendre le build

> À coller dans Claude Code pour démarrer une session de dev.

1. Lis `docs/` en entier. Ne code rien tant que ce n'est pas fait.
2. Ouvre `docs/TASKS.md`. Prends la **première tâche non cochée** de la version
   en cours (commence par v0.1).
3. Avant de coder : résume en 3 lignes ce que tu vas faire et pourquoi.
4. Implémente **une seule** tâche. Petite, testable.
5. Teste-la sur un cas réel.
6. Coche la tâche dans `TASKS.md`, note toute décision dans `DECISIONS.md`.
7. Commit avec un message clair.
8. Reviens vers l'humain pour validation avant la tâche suivante.

Ne pars jamais sur plusieurs tâches en parallèle.
